# git clone

O comando `git clone` copia um repositório remoto para sua máquina.

Uso básico:

```bash
git clone https://github.com/usuario/repositorio.git
```

O `clone` cria uma pasta com todo o histórico do projeto e configura o remoto `origin`.

Quando usar:

- para começar a trabalhar em um projeto existente
- para obter uma cópia local de um repositório do GitHub
