# Common Git Questions

Este documento lista perguntas frequentes em entrevistas sobre Git.

- O que é Git e por que usar?
- Qual a diferença entre `git pull` e `git fetch`?
- Como resolver conflitos de merge?
- O que são branches e por que usá-los?
- Como funciona o controle de versão distribuído?

Pratique responder com exemplos simples de uso no dia a dia.
