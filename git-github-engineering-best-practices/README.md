# Git + GitHub Engineering Best Practices

## Overview / Visão geral

This folder contains a practical guide for Git and GitHub workflows used by engineering teams.

Esta pasta contém um guia prático para fluxos de trabalho de Git e GitHub usados por equipes de engenharia.

## How to use this guide / Como usar este guia

Start with the fundamentals in `01-git-fundamentals/`, then move into branching and collaboration topics.

Comece pelos fundamentos em `01-git-fundamentals/`, depois avance para tópicos de branching e colaboração.
