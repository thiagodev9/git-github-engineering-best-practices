# Repo Structure

Uma boa estrutura de repositório facilita o entendimento do projeto.

Dicas:
- mantenha código, documentação e dados separados
- use pastas claras como `src/`, `docs/`, `tests/`
- evite arquivos soltos sem propósito
- inclua `README.md` e `.gitignore`

Organizar o repositório melhora a navegação e o trabalho em equipe.
