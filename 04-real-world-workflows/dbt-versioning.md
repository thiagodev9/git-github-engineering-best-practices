# dbt Versioning

No dbt, versionar modelos e macros permite controlar a evolução do projeto analítico.

Recomendações:
- use Git para gerenciar modelos e configurações
- evite mudanças grandes sem revisão
- crie branches para novas transformações
- documente atualizações de modelos

Assim você mantém histórico de mudanças e segurança no desenvolvimento de dados.
