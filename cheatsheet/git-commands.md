# Git Commands Cheat Sheet

Comandos úteis para trabalhar com Git:

- `git init`: iniciar repositório local
- `git status`: ver o estado dos arquivos
- `git add <arquivo>`: adicionar ao stage
- `git commit -m "mensagem"`: criar commit
- `git log`: ver histórico de commits
- `git diff`: ver diferenças entre versões
- `git branch`: listar branches
- `git checkout <branch>`: mudar de branch
- `git checkout -b <branch>`: criar e mudar para branch
- `git merge <branch>`: juntar outra branch
- `git pull origin main`: atualizar branch local
- `git push origin main`: enviar mudanças para remoto
- `git reset HEAD <arquivo>`: retirar arquivo do stage
- `git restore <arquivo>`: descartar mudanças locais
