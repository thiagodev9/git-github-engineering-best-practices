# git add + git commit

Os comandos `git add` e `git commit` são usados para registrar mudanças no histórico do projeto.

### git add

Prepara arquivos para o próximo commit.

```bash
git add arquivo.txt
```

Para adicionar todos os arquivos modificados:

```bash
git add .
```

### git commit

Cria um ponto no histórico com as alterações preparadas.

```bash
git commit -m "Mensagem clara de commit"
```

Dica:
- use mensagens curtas e descritivas
- faça commits pequenos e frequentes
