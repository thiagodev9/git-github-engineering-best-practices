# Projeto JornadaDados

Este repositório contém um projeto em Python.

## Arquivos

- `main.py`: arquivo principal do projeto.

## Como usar

1. Instale o Python.
2. Execute:
   ```bash
   python main.py
   ```

## Git

- Inicialize o repositório local com `git init`
- Adicione arquivos com `git add .`
- Faça commit com `git commit -m "Primeiro commit"
- Conecte ao GitHub com `git remote add origin <URL>`
- Envie para o GitHub com `git push -u origin main`
